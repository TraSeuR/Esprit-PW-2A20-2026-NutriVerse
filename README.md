##  Description du Projet

**NutriVerse** est une application web développée dans le cadre du module **Programmation Web (PW)** à **Esprit School of Engineering** (Année universitaire 2025–2026).

Elle permet aux utilisateurs de gérer leur alimentation, suivre leurs apports nutritionnels et adopter un mode de vie sain grâce à une interface simple et intuitive.

---

##  Objectifs

*  Promouvoir une alimentation saine et durable
*  Permettre la planification de repas équilibrés
*  Réduire le gaspillage alimentaire
*  Suivre les apports nutritionnels et les calories
*  Adapter l’alimentation aux besoins spécifiques (notamment sportifs)
*  Encourager la consommation de produits locaux

---

##  Fonctionnalités

*  **Gestion des utilisateurs** : création, authentification et gestion de profil
*  **Gestion des recettes** : consultation, ajout et recommandation de recettes saines et durables
*  **Planification des repas** : génération de menus personnalisés selon les préférences et besoins
*  **Suivi nutritionnel** : suivi des calories et des nutriments consommés
*  **Produits locaux** : découverte et consultation de produits proposés par des producteurs locaux
*  **Gestion des commandes** : possibilité de commander des produits
*  **Sport & nutrition** : proposition de programmes sportifs personnalisés avec des régimes alimentaires adaptés
*  **Recommandation intelligente** : suggestions basées sur les préférences, besoins et ingrédients disponibles

---


##  Architecture du Projet

Le projet suit une architecture web classique :

* **Frontend** : Interface utilisateur (HTML, CSS, JavaScript)
* **Backend** : Logique métier (PHP)
* **Base de données** : Stockage des informations (MySQL)

##  Installation

1. Cloner le repository :

```bash
git clone https://github.com/TraSeuR/Esprit-PW-2A20-2026-NutriVerse.git
```

2. Accéder au dossier :

```bash
cd Esprit-PW-2A20-2026-NutriVerse
```

3. Configuration :

* Avec **XAMPP** :

  * Déplacer le projet dans `htdocs`
  * Lancer Apache et MySQL
